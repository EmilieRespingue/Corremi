#### Corremi
Un petit outil pour aider les gens qui écrivent toujours les mêmes phrases. Listez simplement les phrases récurrentes dans le fichier texte et cochez les ensuite dans le logiciel qui les assemblera automatiquement. Utilisé à la base pour aider à corriger des contrôles.


##### Configuration requise
Corremi utilise Java 8 ou supérieur qui peut être installé ici: https://www.java.com/fr/download/manual.jsp


##### Comment ça marche
Dans le dossier data il y a le fichier corremi.txt. Dedans vous pouvez définir les titres et les commentaires que vous utilisez fréquement. Si vous avez besoin d'ajouter un retour à la ligne mettez une balise <br>. Si vous notez un contrôle vous pouvez mettre une balise <> ou il y aura une note partielle et une balise <total> ou la note totale sera.
Quand vous lancez Corremi en double cliquant sur Corremi.jar chaque titre sera un onglet et chaque ligne sera une checkbox en dessous. Cocher une checkbox ajoutera la ligne au texte du dessous. Vous pouvez éditer la ligne, ce qui cochera la checkbox automatiquement. Mettre un nombre dans les balises <> affichera le nombre et mettre à jour le <total>.
Quand le texte vous convient vous pourrez l'enregistrer dans un fichier ou le copier dans le presse-papier grâce aux boutons du bas.
